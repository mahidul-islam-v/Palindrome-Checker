# Paal In D' Rome

Your Italian paal!! A simple yet interesting tool to check if a word is palindrome. Search your vocabulary and discover words that read the same forwards and backwards!

---

## 📖 What is a Palindrome?

A **palindrome** is a word that is spelled exactly the same forwards and backwards.

### Examples:

- **refer** — r-e-f-e-r (same both ways)
- **wow** — w-o-w
- **racecar** — r-a-c-e-c-a-r

---

## ✨ Features

- **Dictionary Validation** — Checks if your input is a valid English word using an API
- **Palindrome Detection** — Analyzes the string to determine if it's a true palindrome
- **Instant Feedback** — Get immediate results as you search
- **Simple & Intuitive** — Easy-to-use interface

---

## 🔍 How It Works

1. Enter a word into the application
2. Paal queries a dictionary API to verify it's a valid English word
3. The input string is reversed and compared to the original
4. Displays result whether the word is a palindrome or not

**Algorithm:**

- User input is normalized
- String reversal is performed
- Original vs. reversed comparison determines palindrome status
